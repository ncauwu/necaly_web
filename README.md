# necaly_web
website weekly planner yang imut imut (tugas pbw)

necaly/

├─ config/koneksi.php   → nyambungin PHP ke database MySQL

├─ index.php            → halaman awal

├─ register.php         → daftar akun baru

├─ login.php            → masuk pakai email + password

├─ logout.php           → keluar / hapus sesi

├─ dashboard.php        → halaman utama (list folder bulan)

├─ dashboard.css        → tampilan dashboard

├─ note.php             → halaman buka folder (mood, schedule, to-do, notes, bintang week)

├─ note.css             → tampilan note

└─ style.css            → tampilan login & register
